# HTML-CSS-JS-Portfolio

这是我的前端基础练习仓库，主要用来存放使用 **HTML + CSS + JavaScript 三件套** 实现的前端作品。

仓库里的项目分为两类：
- ✨ 原创练习：基于自己的想法实现的小页面、交互效果
- 📚 临摹作品：参考优秀设计/教程实现的页面，重点用于练习布局、响应式和原生JS交互

---

所有代码均为手动实现，过程中会根据自己的理解做细节调整与优化。我希望通过这些练习不断夯实前端基础，在实践中持续进步。
